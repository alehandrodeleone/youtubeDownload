GET Запросы(прописаны в requests.http)

Для получения информации о всех записях в базе данных в запросе используйте префикс list_api

Пример:

GET http://127.0.0.1:8000/list_api

Content-Type: application/json

GET Запросы(поиск по id)

Для получения информации о конкретных записях в базе данных в запросе используйте префикс list_api/id/

Пример:

GET http://127.0.0.1:8000/list_api/2/

Content-Type: application/json

CRUD Запросы

Для использования CRUD запросов используйте префикс api_edit

Внимание! Требуется токен

Пример:

POST http://127.0.0.1:8000/api_edit/

Authorization: Token 9249d910c1fba165014b9ac763116fa8c7e9cd3a

Content-Type: application/json
